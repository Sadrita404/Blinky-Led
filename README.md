# Blinky-Led

I built a simple LED Blinker device that makes a light flash on and off automatically. While many people start these projects on a plastic plug-in board, I took mine a step further by designing and making a custom Printed Circuit Board (PCB). This makes the project much more durable, compact, and professional-looking. It’s a great example of a basic timing circuit put into a permanent, finished form.

---

## How I Built It
To bring this project to life, I followed a standard engineering workflow:

Circuit Design: I started by creating a schematic diagram to map out how the battery, resistors, capacitors, and transistors (or a timer chip) would interact to create the blinking effect.

PCB Layout: Using design software, I translated the schematic into a physical board layout, routing the "traces" (copper paths) where the electricity would flow.

Fabrication: I had the PCB manufactured, giving me a solid blue board with precise pads for each part.

Assembly (Soldering): I placed the physical components—the LEDs, resistors, and power source—onto the board and used a soldering iron to permanently bond them to the circuit.

Testing and Power-on: Finally, I connected a power source to ensure the connections were solid. The project was a success: the LED glows brightly and pulses at the intended speed.
