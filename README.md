# Blinky-Led

I built a simple LED Blinker device that makes a light flash on and off automatically. While many people start these projects on a plastic plug-in board, I took mine a step further by designing and making a custom Printed Circuit Board (PCB). This makes the project much more durable, compact, and professional-looking. It’s a great example of a basic timing circuit put into a permanent, finished form.
